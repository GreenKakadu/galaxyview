Groups Table

The Groups Table lists information about groups of ships. Each column represents an attribute of a group of ships, as follows:

  • Group number
  • Ship type (name)
  • Number of ships
  • Number of attacks
  • Speed (base speed * drive technology)
  • Weapon strength (base weapon * weapon technology)
  • Shield strength (base shield * shield technology)
  • Cargo capacity (base cargo capacity * cargo technology)
